## 一、项目概述

### 1.1 项目信息
- **项目名称**: GoDaemon
- **项目定位**: 类大漠插件后台自动化框架
- **开发语言**: Go 1.21+
- **目标平台**: Windows 7/10/11

### 1.2 核心功能
| 功能模块 | 实现状态 | 说明 |
|----------|----------|------|

| 后台截图 | ✅ 已完成 | GDI截图，支持窗口最小化 |
| 键鼠操作 | ✅ 已完成 | PostMessage后台操作 |
| 找图功能 | ✅ 已完成 | 模板匹配，相似度可调 |
| 找色功能 | ✅ 已完成 | RGB颜色匹配，容差可调 |

## 七、总结
