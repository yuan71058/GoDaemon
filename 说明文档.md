## 一、项目规划

### 1.1 项目定位
基于 Go 语言开发一套开源、高性能、纯后台的 Windows 自动化框架，功能完全对标大漠插件核心功能。

### 1.2 核心目标
- 64位 DLL 编译
- 纯后台操作（窗口最小化、遮挡、隐藏均可正常执行）
- 跨语言调用（易语言/Python/C#/按键精灵）

### 1.3 核心功能模块
| 模块 | 功能描述 | 对应大漠功能 |
|------|----------|--------------|
| 窗口管理 | 窗口绑定、句柄获取、状态判断 | BindWindow、GetWindowHandle |
| 后台截图 | 窗口最小化/遮挡时截图 | Capture、CaptureBgr |
| 键鼠操作 | 后台鼠标移动、点击、键盘输入 | MoveMouse、LeftClick、KeyPress |
| 图色识别 | 找图、找色、多点比色 | FindPic、FindColor、FindMultiColor |
| OCR识别 | 本地离线文字识别 | Ocr、FindStr |
| DLL导出 | 32/64位DLL供外部调用 | 完整API导出 |

### 1.4 技术栈
- **开发语言**: Go 1.21+
- **Windows API**: github.com/lxn/win
- **图像处理**: gocv.io/x/gocv (OpenCV 4.8.0)
- **OCR识别**: github.com/otiai10/gosseract (Tesseract)
- **编译工具**: MinGW-w64 12.2.0

## 二、实施方案

### 2.1 开发流程（6A工作流）
1. **Align（对齐）**: 需求分析、边界确认
2. **Architect（架构）**: 系统架构设计
3. **Atomize（原子化）**: 任务拆分
4. **Approve（审批）**: 计划审查
5. **Automate（执行）**: 代码实现
